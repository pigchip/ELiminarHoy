Gestión de Alumno

  • Para crear un alumno, ingrese los datos y presione el botón "Crear/Actualizar".
  • Para actualizar un alumno, ingrese el id del alumno y presione el botón "Cargar Datos". Después, actualice los datos y presione el botón "Crear/Actualizar".
  • Para eliminar un alumno, ingrese el id del alumno y presione el botón "Eliminar".
